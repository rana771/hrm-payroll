# Registration and Login Example with Spring Security, Spring Boot, Spring Data JPA, MYSQL, JSP


## Prerequisites
- JDK 1.7 or later
- Maven 3 or later

## Stack
- Spring Security
- Spring Boot
- Spring Data JPA
- Maven
- JSP
- HSQL

## Run
```mvn clean spring-boot:run```

##Install Lombok
-Go to File->Settings->Plugins then Search lombok
-Get lombok and install
-restart intellij
